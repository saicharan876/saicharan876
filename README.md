# 🚀 Featured Projects

### 🔬 [QuantumRAG](https://github.com/saicharan876/QuantumRAG)
Agentic RAG assistant for quantum computing research — multi-agent Planner-Retriever-Responder pipeline with NeMo Guardrails, Qdrant semantic search, and FlashRank reranking, served via Portkey LLM gateway with Logfire observability.

### 🏛️ [EpicGeo-ML](https://github.com/saicharan876/EpicGeo)
Spatial ML pipeline predicting archaeological sites from ancient epics — 112,000+ grid points, Random Forest/XGBoost, NLP-mined historical text features, AUC 0.81, validated with Spatial K-Fold Cross-Validation.

### 🌍 [TravelBuddy](https://github.com/saicharan876/Travel-Buddy)
Social travel matching platform using spatiotemporal trip overlap, real-time geolocation (Leaflet.js, OpenStreetMap, Nominatim API), JWT auth, and Socket.IO group messaging.
